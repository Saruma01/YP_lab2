Цель лабораторной работы: изучить основные подходы к сортировке массивов, а также способы визуализации процесса сортировки.
Задачи:
- изучить методы сортировки массивов;
- освоить понятия устойчивых и неустойчивых сортировок;
- изучить возможность визуализации процессов, используя выделение цветом и форматированный вывод, покадровый вывод (задержку экрана);
Задание: 
- реализовать 3 метода сортировки массива (пузырьковая, расческой, быстрая) с визуализацией процесса. 
- предусмотреть возможность задания размерности массива (10-15 элементов) и границ диапазона значений элементов. 
- визуализация должна отображать сравнения и перемещения элементов цветом или другим способом, с использованием задержки экрана.
